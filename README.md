# Repso Drive – Frontend Demo

Proyecto frontend desarrollado en **Angular** como demostración visual de una aplicación tipo Drive.

 **Nota:**  
Este proyecto es una **demo de interfaz**.  
No cuenta con backend activo ni autenticación real.
